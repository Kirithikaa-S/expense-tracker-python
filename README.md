## Project Description
This is a console-based Expense Tracker developed using Python and MySQL. It helps users manage their daily expenses by storing and analyzing expense records.

## Features
- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- Search Expense
- Category Summary
- Monthly Summary

## Technologies Used
- Python
- MySQL
- mysql-connector-python
- VS Code
- GitHub

## How to Run
1. Install Python
2. Install mysql-connector-python
3. Create the MySQL database and expenses table
4. Update database credentials in db.py
5. Run:

```bash
python main.py
```
